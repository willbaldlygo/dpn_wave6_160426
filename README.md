# AI Bootcamp Progression Dashboard

This dashboard provides a qualitative extraction of learner sophistication and Bloom's taxonomy ceilings across two primary frameworks:
- **Usage Reflections**: Weekly narrative arcs of AI adoption.
- **Milestone Progression**: Assessment of growth from baseline to course outcome.

## Access
This dashboard is protected by a **Static Lock**. 
- Open the site via the [GitHub Pages link](https://willbaldlygo.github.io/dpn_wave6_160426/).
- Enter the organization password when prompted.
